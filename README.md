## 📌 주요 구현 내용

1. 카카오 맵 API 활용하여 지도 구현
2. 클러스터러 활용한 데이터 시각화 (카카오맵)
3. 공공 API 데이터를 이용한 실제 병원 데이터 표출
4. Spring Security를 활용한 로그인, 로그아웃, PW 암호화
5. 메일을 통한 임시패스워드 전송 기능
6. 게시판 CRUD, 조회수 기능
7. 댓글 및 좋아요 기능
8. 파일 업로드 기능
9. 게시글, 병원 목록 별 페이징 처리
10. 예약 기능
11. 도메인 적용하여 AWS EC2 서버에 배포
